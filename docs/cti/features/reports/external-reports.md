# External Reports

**External reports** are intelligence reports published by other organizations (consultancies, vendors, CERTs, OSINT sources, etc.) that have been parsed and integrated into the Intelligence database.

For each external report, our teams:

* Parse the text and metadata, identifying indicators, threat actors, malware, tactics, and related context.
* Create structured intelligence objects (observables, threat actors, malware, campaigns, etc.) linked to the report.
* Enrich the report with TLP, sectors, countries, and confidence, so it fits into the same model as FLINTs.

Using external reports, you can:

* Discover intelligence produced by external sources while staying inside the Sekoia ecosystem.
* Pivot from the report to associated observables and entities.
* Use the same STIX‑based model and filters as for FLINTs and other intelligence objects.
