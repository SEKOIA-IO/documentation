# Export STIX objects

Use **Export** to extract STIX objects from any object page. This lets you reuse intelligence outside the platform, share it with other teams, or ingest it into downstream tools.

### What you can export <a href="#what-you-can-export" id="what-you-can-export"></a>

From an object page, the export action lets you export related STIX objects as structured data. You can choose:

* **All threat context**.
* **Selected categories**.

### Export formats <a href="#export-formats" id="export-formats"></a>

You can export the data in one of the following formats:

* **CSV**.
* **JSON lines**.
* **Text**.

### How it works <a href="#how-it-works" id="how-it-works"></a>

1. Open an object page.
2. Click **Export**.
3. Choose whether to export all threat context or only selected categories.
4. Select the output format.
5. Click **Export** to download the data.

### When to use it <a href="#when-to-use-it" id="when-to-use-it"></a>

Use export when you want to:

* Reuse intelligence in another tool.
* Share object context with analysts or stakeholders.
* Archive structured threat data for later analysis.
* Copy threat context into scripts, reports, or workflows.



{% hint style="info" %}
The export feature is available directly from each object, so you can extract only the context you need without leaving the page.
{% endhint %}
