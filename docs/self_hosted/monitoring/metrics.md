# Platform metrics

The platform exposes technical metrics to monitor the health of the underlying Kubernetes cluster and microservices.

## Key performance indicators
| Metric | Description |
| :--- | :--- |
| **Node Health** | CPU, memory, and disk utilization per node. |
| **Pod Status** | Availability and restart counts for microservices. |
| **Ingestion Volume** | Daily GB ingested vs. licensed capacity. |
| **Detection Latency** | Time taken between event ingestion and rule matching. |
