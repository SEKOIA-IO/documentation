# Release Note 0.1.0

This is the initial release of the Sekoia Self-Hosted platform MVP.

## New features
* **Air-gap support**: Full deployment capability in restricted environments.
* **Deployment CLI**: Introduction of the unified orchestration tool.
* **Signed Artifacts**: Verifiable and tamper-evident delivery model.

## Technical foundation
* **Kubernetes stack**: Based on K3s.
* **OS Support**: Certified for Debian 11.
* **AI SOC Core**: Inclusion of built-in dashboards and diagnostic tools.
