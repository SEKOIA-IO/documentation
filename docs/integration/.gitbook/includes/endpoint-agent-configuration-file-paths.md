---
title: Endpoint agent configuration file paths
---

Use the following path for your operating system.

{% tabs %}
{% tab title="Windows" %}
```shell
C:\Windows\System32\config\systemprofile\AppData\Local\Sekoia.io\EndpointAgent\config.yaml
```
{% endtab %}

{% tab title="Linux" %}
```shell
/etc/endpoint-agent/config.yaml
```
{% endtab %}

{% tab title="MacOS" %}
```shell
/etc/endpoint-agent/config.yaml
```
{% endtab %}
{% endtabs %}
