### Overview

Trend Micro Vision One is an extended detection and response (XDR) platform that enhances threat detection, investigation, and response across multiple security layers. It provides a centralized view for improved security posture and faster threat remediation. This integration will ingest Workbench Alerts from Trend Micro Vision One.

* **Supported environment**: SaaS
* **Detection based on**: Alerts
* **Supported application or feature**:
  * Alerts
